# HW02

## Run Tests
1. Переместиться в директорию hw02
2. python -m pytest -v src/tests/test_parser_json.py

## Check Coverage
1. Переместиться в директорию hw02
2. python -m pytest --cov=src/main

## Run Pylint
1. pylint src/main/parser_json.py
