# HW01

## Run Tests
1. Переместиться в директорию hw01
2. python -m pytest -v src/tests/test_tic_tac_game.py

## Check Coverage
1. Переместиться в директорию hw01
2. python -m pytest --cov=src/main

## Run Pylint
1. pylint src/main/tic_tac_game.py
